Stop Suap! Berbagi cerita tentang pengalaman yang berhubungan dengan korupsi, kolusi, dan nepotisme.

## Dependencies

- Ruby version 2.2.3
- Rails version 4.2.1
- PostgeSQL
- Mailcatcher (for email server in development)
